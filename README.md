# Marvel Characters App

React application that allows users to browse Marvel characters using the Marvel public API.

Users can search for characters and view detailed information.

## Tech Stack

React  
JavaScript  
REST API  
CSS  

## Features

- search Marvel characters
- display character cards
- pagination
- view character details

## API

This project uses the Marvel public API.

https://developer.marvel.com
